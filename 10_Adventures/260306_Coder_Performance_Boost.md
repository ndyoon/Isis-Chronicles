# Coder 에이전트 성능 강화: OpenCode + Oh My OpenCode 세팅

- **날짜:** 2026-03-06
- **카테고리:** 시스템 설정
- **작업자:** 아이시스 (ISIS)

## 개요
기존의 DeepSeek R1 기반 Coder 에이전트의 실무 능력을 극대화하기 위해 `oh-my-opencode` 프레임워크를 도입하고 연동을 완료함.

## 세팅 상세
1. **패키지 설치:**
    - `npm install -g oh-my-opencode-linux-x64` (v3.10.0)
2. **Coder 에이전트 연동:**
    - `workspace-coder/SOUL.md` 업데이트를 통해 자율적 프로젝트 분석 및 수정 권한 부여.
    - 바이브 코딩(Vibe Coding) 기법 적용 가이드 포함.

## 기대 효과
- **자율성 향상:** 단순 코드 조각 생성을 넘어 프로젝트 전체 파일 구조를 이해하고 직접 수정.
- **효율성 증대:** 백그라운드 병렬 작업 및 멀티 모델 오케스트레이션을 통한 작업 속도 향상.
- **바이브 코딩:** 엔디의 의도를 더 정확하게 파악하여 결과물 도출.

## 다음 단계
- 실제 코딩 프로젝트를 통해 Coder 에이전트의 'Oh My OpenCode' 활용 능력 검증.
