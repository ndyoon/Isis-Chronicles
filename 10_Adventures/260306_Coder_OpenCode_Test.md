# Coder 에이전트 OpenCode 실전 테스트: 파이썬 대시보드 제작

- **날짜:** 2026-03-06
- **테스트 항목:** OpenCode `run` 기능을 활용한 자율 스크립트 생성 및 테스트
- **작업자:** 아이시스 (Isis) & Coder 에이전트

## 테스트 개요
엔디의 요청에 따라 현재 폴더에 '오늘의 날씨(서울)'와 '비트코인 시세(업비트)'를 보여주는 파이썬 대시보드 제작 시연.

## 진행 과정
1. **환경 구축:** `opencode-ai` 패키지 설치 및 환경 설정.
2. **모델 선택:** 
    - DeepSeek R1/V3 및 Kimi API의 일시적 오류(잔액 부족/만료)로 인해 `opencode/minimax-m2.5-free` 모델로 우회하여 진행.
3. **자율 작업:**
    - `opencode run` 명령어를 통해 `dashboard.py` 파일 생성.
    - LSP(언어 서버) 진단을 통해 잠재적 오류(`None` 타입 처리) 자율 수정.
    - `python3` 명령어를 사용하여 스크립트 실행 및 결과 확인.

## 결과물 (`dashboard.py`)
- **날씨 데이터:** Open-Meteo API (비인증) 활용.
- **비트코인 데이터:** Upbit API 활용.

## 시사점
- `opencode`의 자율적인 에러 수정(Self-healing) 능력이 확인됨.
- 멀티 모델 오케스트레이션을 통해 특정 API 장애 시에도 유연하게 대처 가능함을 증명.
