# Zeekr 7X 기능 구현 기획안

## 🚗 개요
지커 7X(Zeekr 7X)의 인포테인먼트 시스템과 아이시스(Isis) AI 에이전트를 통합하여 차량 내에서도 끊김 없는 비서 서비스를 제공하는 기능 구현 계획.

## 🛠️ 기술적 기반
- **Hardware:** Qualcomm Snapdragon 8295 칩셋 (고성능 AI 연산 및 그래픽 처리 최적화).
- **Software:** Zeekr OS (Android Automotive OS 기반). 웹 표준 기술(HTML5, PWA) 및 API 통신 수용성 높음.

## 💡 주요 구현 시나리오
### 1. 웹 기반 통합 대시보드 (Mission Control - Car Mode)
- 차량 내 브라우저를 통해  접속.
- 아바타 상주 및 실시간 자산(BTC), 시스템 상태 브리핑.
- 차량 디스플레이 크기에 맞춘 반응형 '카 모드' 레이아웃 적용.

### 2. PWA 기반 독립 앱 설치
- Progressive Web App 기술을 활용하여 차량 시스템에 '아이시스'를 독립 앱으로 설치.
- 모바일 앱과 유사한 네이티브 사용자 경험 제공.

### 3. 음성 비서 고도화 연동
- 차량 마이크/스피커 시스템과 저의 음성 엔진(TTS/STT) 통합.
- 주행 중 핸즈프리 자산 브리핑 및 일정 관리 수행.

## 🔐 보안 및 연결성
- **Connectivity:** 차량의 5G 네트워크를 통한 원격 서버 상시 연결.
- **Security:** 차량 기기 인증(Device Identity)을 통한 안전한 접근 제어.

## 📅 실행 로드맵
1. 미션 컨트롤 대시보드 내 'Car Mode' UI 프로토타입 제작.
2. 차량 브라우저 환경에서의 보안 토큰 인증 체계 검증.
3. 2026년 국내 출시 시점에 맞춘 최종 배포 준비.
